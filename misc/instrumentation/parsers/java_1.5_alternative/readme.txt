This parser is based on Sreenivasa Viswanadha Java 1.5 parser. I correct some bugs and added AST generation suport. The AST generated by this parser is not like AST automatic generated by jjTree. It is like Eclipse JDT Dom, where the nodes are more optimized e the generated tree has only de leef nodes. I added suport for two kind of visitors, both with generics suport.


Source distribution:
--------------------

src-folder: parser
packages:
	java.parser: the jj file is here
	java.parser.ast: the ast nodes start here
	java.parser.ast.body: class body member nodes 
	java.parser.ast.expr: expression nodes 
	java.parser.ast.stmt: statement nodes 
	java.parser.ast.type: type nodes 
	java.parser.ast.visitor: visitor interfaces and DumpVisitor implementation
	
src-folder: test
packages:
	java.parser.ast.test: a simple jUnit to test the tree and the DumpVisitor 

	
If you have found any bug or find a feature enhacement, please contact me.


Sample usage
------------

CompilationUnit cu = JavaParser.parse(new FileInputStream("test.java"));
System.out.println(cu.toString());
